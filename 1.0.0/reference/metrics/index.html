    deepsentinel.metrics¶

    metrics ¶

    DeepSentinel SDK performance metrics and monitoring.

    This module provides comprehensive performance monitoring, metrics collection, and reporting utilities for tracking SDK performance and usage patterns.

    Classes¶

    MetricsCollector ¶

    MetricsCollector(enabled=True, window_size=1000, retention_minutes=60)

    Comprehensive metrics collector for SDK performance monitoring.

    This class collects and aggregates performance metrics including: - Request performance and timing - Token usage and costs - Cache effectiveness - Provider-specific metrics - System health indicators

    Attributes:

    Name Type Description
    enabled

    Whether metrics collection is enabled
    window_size

    Size of the rolling window for metrics
    logger

    Structured logger

    Parameters:

    Name Type Description Default
    enabled bool

    Whether to collect metrics

    		 True
    window_size int

    Size of rolling window for recent metrics

    		 1000
    retention_minutes int

    How long to retain detailed metrics

    		 60
